Wakey - 专注深度工作的番茄钟

一款为创作者、开发者和知识工作者打造的极简番茄钟与专注计时器，专为 macOS 设计。

Wakey 帮助你把一天拆分为有节奏的专注冲刺与刻意休息。没有信息流，没有社交元素——只有一个安静、可靠、始终在旁的计时器。

核心功能
• 番茄工作法节奏——专注时段搭配短休息与长休息
• 细腻不打扰的通知提醒
• 简洁清爽的 macOS 原生界面
• 轻量级设计，占用资源极少

为什么选择 Wakey？

许多效率应用试图「全都做」——任务管理、日历、习惯追踪……结果反而让人分心。Wakey 只做一件事：帮你看见当下这段专注时间，并安静地陪你完成它。

适合人群
• 需要长时间深度工作的开发者与设计师
• 备考或做项目的学生
• 作者、研究者与内容创作者
• 喜欢安静、不花哨效率工具的任何人
