## ✨ 功能特性

Wakey 致力于成为你得力的 macOS 状态栏助理，提供以下核心能力：

- **防休眠控制**：一键阻止系统休眠，支持灵活的模式切换与时长设置，确保重要任务不因息屏而中断。
- **状态栏助理**：在菜单栏常驻，无需切换窗口即可快速访问各项功能。
- **模块化设计**：采用插件化架构，易于扩展更多功能模块。
- **个性化体验**：根据个人需求在设置中灵活开启或关闭各项功能，打造专属的助理界面。

## 📋 系统要求

- macOS 13.0+
- Xcode 15.0+
- Swift 5.9+
